### Aircraft Force Balance
In cruise:
$$W=L=\frac{1}{2}\rho V^{2}S C_{L}$$
$$T=D=\frac{1}{2}\rho V^{2}S C_{D}$$
### Aerodynamic Efficiency Index
We can calculate Aerodynamic Efficiency to be:
$$\eta_{aero}=\frac{Lift}{Drag}=\frac{C_{L}}{C_{D}}$$
Therefore if we want to increase the aerodynamic efficiency we either need to increase the $C_{L}$, decrease the $C_{D}$ or both.

#### Reducing Drag
We can decrease the sources of drag in different ways:

| Reducing $D_0$                        | Reducing $D_L$                         |
| ------------------------------------- | -------------------------------------- |
| Streamlined configuration             | Increase Aspect Ratio                  |
| Smaller LGs, conformal antennae       | Increase $b_{eff}$ for the same L or W |
| Smooth surface finish                 |                                        |
| Low $S_{wet}$                         |                                        |
| Technologies to delay flow separation |                                        |
Note that $b_{eff}$ is the effective span i.e. $span+winglets$. So:
$$D_{L}=(L/b_{eff})$$
